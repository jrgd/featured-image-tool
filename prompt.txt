# PROMPT

this an html/css page + js
this tool will create a high resolution image based on data input on the page; both the data and the image are visible
user will input a maximum of 4 line of text
the tool will insert the text with a custom font
one line at atime, each line at a random angle
without the text overlapping
there a couple of controls on the left hand side:
- image size in pixel (w / h)
- text strings input (x4, one per line of text)
- each line of text gets a random angle, that is editable
- text color; clicking on the color swatch let's you edit it hls gradients
- background-color
- font-size (1 size fits all for now)
the image is then save-able; i imagine this would be a canvas image generated on the client side (js); there should be no server-side ops
interface should be minimalist (10px font size, monospace, B/W, no shadow, 1px solid black border, 5px margin)
